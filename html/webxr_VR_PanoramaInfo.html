VR Panorama (WebXR)


This example uses WebXR to create a Virtual Reality Panorama.

The example is best suited for use with mobile devices. After clicking on the "Open Example" button below, press the "Enter VR" button. This button is only available if your mobile device is ready for WebXR. You will have a better experience when using a HMD (Google Cardboard or similar).

Note: To test this example on a computer you have to install the WebXR API Emulator extension and select "Google Cardboard" or another HMD.

Further information about webxr can be found here.
