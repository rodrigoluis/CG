Geometries - Basic


There are many ways to work with geometries in threejs. The easiest way to start working with geometries is by using the available 3D primitives. You can easily create cubes, spheres, torus and other basic geometries just by setting some basic parameteres of each object. See the example below and change the object to be rendered through the drop down list available.

Open in a new tab

Setting parameters

Just to illustrate how the creation of objects in threejs is easy, see the code below to understand which parameters must be provided to create an sphere. Basically you have to inform the radius of the sphere, number of horizontal and vertical segments. You can use default parameters but at least these three are suggested to be set. The objectMaterial used in this example will be explored in further examples.

var radius = 3.0; var widthSegments = 20; var heightSegments = 20; var geometry = new THREE.SphereGeometry(radius, widthSegments, heightSegments); var sphere = new THREE.Mesh(geometry, objectMaterial); scene.add(sphere);
