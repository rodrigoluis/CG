Geometric Transformations 1


This project covers the most common geometric transformations - translation, rotation and scale. All these transformations in this project are applied to a cube and controlled by the keyboard. Try the above project following the instructions presented on canvas.

Applying transformations

As we control the geometric transformations using the keyboard, all functions are applied in the KeyboardUpdate function.

function keyboardUpdate() { (...) if ( keyboard.pressed("left") ) cube.translateX( -1 ); if ( keyboard.pressed("right") ) cube.translateX( 1 ); if ( keyboard.pressed("up") ) cube.translateY( 1 ); if ( keyboard.pressed("down") ) cube.translateY( -1 ); if ( keyboard.pressed("pageup") ) cube.translateZ( 1 ); if ( keyboard.pressed("pagedown") ) cube.translateZ( -1 ); if ( keyboard.pressed("A") ) cube.rotateOnAxis(rotAxis, angle ); if ( keyboard.pressed("D") ) cube.rotateOnAxis(rotAxis, -angle ); if ( keyboard.pressed("W") ) { scale+=.1; cube.scale.set(scale, scale, scale); } if ( keyboard.pressed("S") ) { scale-=.1; cube.scale.set(scale, scale, scale); } }

As can be observed through the fragment code presented, in threejs one way to work with geometric transformations is through the methods available to any Object3D object. In this example we use the translateX-Y-Z methods to translate, the rotateOnAxis to rotate the cube along an axis in its space (i.e., object space) and the scale.set to change it's scale attribute.

Those methods are very useful but they do not allow a full control over how those transformations are applied. What if you want to control the order in which the transformations area applied? What if you want to manually set your transformation matrix? We will see how it works in the next project.
