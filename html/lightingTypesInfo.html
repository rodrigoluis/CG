Types of lights


In Computer Graphics there are several types of lights that can be used by the developers. Threejs has also several types of lights available and in this project we will explore three of them: Spotlight, Directional light and point light.

In the project below, you can change between these lights and see how they behave in the scene.

Open in a new tab

Main Differences

As can be seen in Figure 1 we have important visual differences between these three light types.

When using Spotlights, we have to set the angle of the camera, its position and, if necessary, the target (where the light points). By default the target is (0,0,0). Additionaly we can set maximum range of the light, the percentage of the spotlight cone that is attenuated due to penumbra and the decay of the light.

var spotLight = new THREE.SpotLight("rgb(255,255,255)"); spotLight.position.copy(new THREE.Vector3(1.7, 0.8, 1.1)); spotLight.angle = degreesToRadians(40); spotLight.castShadow = true; spotLight.decay = 2; spotLight.penumbra = 0.5; spotLight.name = "Spot Light" scene.add(spotLight);

The Point Light is much simpler, and you can set this light by providing just its position and color. Differently from the spotlight, this type of light will illuminate in all directions. The way the light is constructed is very similar to the way we create spotlights (i.e., create the object, set parameters and add it to the scene).

Finally you have Directional Lights. The main difference from the other two lights is that in this light we have only a direction, like a vector, in which the light lays, not a point in the space. We can use this type of light to emulate the sun, for example. We have to set a position and a target. These two points will form the aforementioned vector. You can see in Figure 1 that this type of light usually generate soft shadows and the process is computationally expensive (you can see details about this shadow generation here).
