VR Teleport (WebXR)


This example explores a teleport function in a WebXR environment. When you press the button of your VR HMD you will see a rectile. It turns from white to orange when you are over an area in which you can be teleported.

The example is best suited for use with mobile devices. After clicking on the "Open Example" button below, press the "Enter VR" button. This button is only available if your mobile device is ready for WebXR. You will have a better experience when using an HMD (Google Cardboard or similar) that has a button, or a joystick that simulates this behavior (a screen touch).

Note: To test this example on a computer you have to install the WebXR API Emulator extension and select a HMD that has a joystick.
