Dynamic vs Static shadows


In this example, you can see the impact on the application's performance by turning off the 'AutoUpdate' parameter of a directional light. This option is especially interesting in applications where shadows don't need to be updated every frame. Check the FPS counter to spot the difference when turning ON this parameter.

This example also presents a secondary directional light to be used with dynamic objects. This light can project shadows in a smaller area of influence to reduce computational consumption. As we create two lights, the intensity of each light is cut by half (i.e., 0.5 for each light).
